# 🧠 Business Logic Vulnerabilities

> PortSwigger Web Security Academy — Business Logic Vulnerabilities

Business logic vulnerabilities arise when an application's intended
workflow can be bypassed due to flawed assumptions made by developers.

## 🔑 Core Concepts Covered

- Trusting unvalidated client-side parameters (price, quantity, role)
- Integer overflow and numeric boundary conditions
- Multi-step workflow manipulation and state skipping
- Coupon and promo code logic bypasses
- Dual-use endpoints with inadequate access controls
- Encryption oracle exploitation
